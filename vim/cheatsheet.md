:Cheat -- Vim Cheet Sheet
==========================
,, -- ファイルを開く
,r -- Grep
,c -- コマンド一覧
,b -- Buffer一覧
^p -- ctrlp

gm -- 行の中心にカーソル移動
ysi{TObj}" -- "で囲む
f -- 文字手前ジャンプ (like j)
- -- 画面単語ジャンプ
{/} -- 段落移動
^t/h -- インデント移動

*/g* -- 単語検索／置換
<Esc><Esc> -- 検索色クリア

^w{+-<>} -- リサイズ
:'<,'>! -- Shellコマンド置換

Programming
------------
<F1> -- doc
<F5> -- Single Compile
<F6> -- :TestNearest
<F7> -- :TestFile
<F8> -- :TagbarToggle
<C-k> -- Toggle Comments
,# -- :ShebangInsert

### coc
gd -- Go to Definition
gr -- References
<F2> -- Rename
<C-f> -- Format
,lc -- Command List
